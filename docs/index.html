Tentang Buku Toko

Buku Toko adalah aplikasi pencatatan keuangan dan stok barang mobile yang berjalan secara lokal di perangkat Anda. Dibuat dengan arsitektur tangguh untuk memberikan performa instan tanpa latensi server.

React Native Expo SDK SQLite Native Google Sheets Two-Way Sync
100% Offline-First
Data disimpan langsung di SQLite lokal perangkat. Sangat aman dan bebas lag.
Cloud Backup Sync
Sinkronisasi asinkron dua arah ke Google Sheets untuk backup cloud opsional.

Fitur Unggulan

Pencatatan keuangan & inventori toko yang dikemas dalam antarmuka seluler yang modern dan dinamis.

Status Keamanan Stok

Bar progres dinamis dengan klasifikasi status stok yang presisi (Habis, Kritis, Menipis, Aman) dan kode warna visual (Merah, Amber, Kuning, Hijau).

Purchase Order (PO) & Analitik

Kelola pesanan barang ke supplier, estimasi ready, lacak status PO (Progress/Done/Cancelled), visual spend leaders, serta jadwal barang masuk.

Keuangan & Kas Toko

Buku kas keuangan sederhana (General Ledger) untuk memantau pemasukan dan pengeluaran secara lengkap beserta log perubahan data.

AI Business Consultant

Fitur penghitung persentase Margin Profit Bersih offline dari omzet penjualan riil Anda lengkap dengan rekomendasi dan tips bisnis taktis.

Sync Google Sheets

Pembuatan otomatis spreadsheet di Google Drive untuk tab Barang, PO, Keuangan, dan Kalkulator dengan background async sync.

Alert Lonceng Pintar

Sistem notifikasi lokal untuk memperingatkan stok habis (0 pcs), pengingat harian kas, alert PO H-1 ready, dan ringkasan keuangan bulanan.

Laporan PDF & CSV

Ekspor riwayat barang dan detail mutasi ke PDF ber-badge kategori/ukuran yang rapi, atau ekspor data ke file CSV instan.

Form Scroll Keyboard

Menggunakan custom FormScrollContainer agar form input baris bawah tidak terhalang oleh keyboard di perangkat Android & iOS.

Panduan Setup & Instalasi

Jalankan aplikasi di komputer lokal atau hubungkan perangkat Anda.

Terminal 
# Masuk ke folder proyek
cd mobile/gudang-app

# Pasang dependensi
npm install

# Jalankan Metro Bundler
npx expo start

Android Emulator

Aktifkan AVD di Android Studio, lalu tekan tombol "a" di terminal untuk instalasi dan sinkronisasi otomatis.

Perangkat Fisik (HP)

Pasang aplikasi Expo Go di HP Anda, kemudian scan kode QR yang tampil di terminal Anda.

Skema Database SQLite

Tabel-tabel relasional yang otomatis di-build saat aplikasi dijalankan pertama kali.

CREATE TABLE items (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  category TEXT,
  price INTEGER NOT NULL DEFAULT 0,      -- Harga Jual
  cost_price INTEGER NOT NULL DEFAULT 0, -- Harga Modal (HPP)
  stock INTEGER NOT NULL DEFAULT 0,
  size TEXT                              -- Ukuran (XS, S, M, L, XL, XXL)
);
CREATE TABLE stock_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  item_id INTEGER NOT NULL,
  type TEXT NOT NULL,                    -- 'IN' | 'OUT'
  qty INTEGER NOT NULL,
  unit_price INTEGER,
  unit_cost INTEGER,
  profit_amount INTEGER NOT NULL DEFAULT 0,
  note TEXT,
  created_at TEXT NOT NULL DEFAULT (datetime('now','localtime')),
  FOREIGN KEY(item_id) REFERENCES items(id)
);
CREATE TABLE purchase_orders (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  supplier_name TEXT,
  orderer_name TEXT,
  item_name TEXT NOT NULL,
  quantity INTEGER NOT NULL DEFAULT 0,
  price INTEGER NOT NULL DEFAULT 0,
  order_date TEXT NOT NULL,
  status TEXT NOT NULL DEFAULT 'PROGRESS', -- 'PROGRESS' | 'DONE' | 'CANCELLED'
  note TEXT,
  close_po_date TEXT,                      -- Format YYYY-MM-DD
  estimated_ready_date TEXT,               -- Format YYYY-MM-DD
  completed_at TEXT,
  created_at TEXT NOT NULL DEFAULT (datetime('now','localtime'))
);
CREATE TABLE bookkeeping_entries (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  amount INTEGER NOT NULL DEFAULT 0,       -- Positif (Masuk), Negatif (Keluar)
  entry_date TEXT NOT NULL,
  note TEXT,
  created_at TEXT NOT NULL DEFAULT (datetime('now','localtime'))
);
